## WHOAMI

Full Stack Developer with **~5 years** designing and shipping end-to-end web and desktop applications. Specialized in **.NET (C#)** and **React / TypeScript**, backed by **PostgreSQL / SQL Server**, **Entity Framework Core**, and deployment on **Azure**. I've shipped software in production for real clients — not side-project demos — and I build AI-assisted dev workflows into my daily process with **Claude Code**.

📍 Villavicencio, Colombia · Remote · Currently studying Systems Engineering (8th semester)
