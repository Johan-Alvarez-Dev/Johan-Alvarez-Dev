### 👋 About me

- 💻 Full Stack Developer with **~5 years of experience** designing and building end-to-end web and desktop applications.
- 🏗️ Specialized in **.NET (C#)** and **React**, with strong command of **PostgreSQL / SQL Server**, **Entity Framework Core**, and deployment on **Azure**.
- 🚀 I've shipped software **in production** for real clients (POS systems, clinical management) and built high-complexity products of my own, including an **AI orchestrator agent**.

- 📍 Villavicencio, Meta, Colombia · Available for **remote** work.

### 🔭 Featured projects

**Diwy — AI orchestrator agent** *(own product)*
AI-assisted development platform for browser and desktop (Windows, Linux, macOS). Orchestrates multiple AI providers via API and integrates external tools (GitHub, Notion, Office, Slack), with workspace creation, live code execution, local model support, and persistent memory.

**DiwyLearn — Course platform** *(own product)*
Course creation and consumption platform with a component-based editor (interactive blocks: video, games, runnable code). Integrates AI models for assisted grading and content generation support.

**POS system for a fast-food chain** *(real client, in production)*
Adopted by ~20 locations, handling 100+ daily orders and a 1,000+ customer base. Real-time inventory with automatic recipe-based stock deduction, kitchen/front-of-house sync via SignalR.
